# Grammatik

Erklärungen zu einzelnen Grammatikpunkten, aufgebaut nach dem Ansatz von Tae Kim:
erst die Logik dahinter, dann die Form, dann Beispiele.

Eine Datei pro Grammatikpunkt, Dateiname nach dem Punkt selbst — z.B. `te-form.md`,
`wa-vs-ga.md`, `negation.md`.
