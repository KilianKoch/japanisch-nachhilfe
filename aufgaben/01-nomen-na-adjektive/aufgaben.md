# Blatt 1 — Nomen und な-Adjektive

**Grammatik:** [Nomen und な-Adjektive](../../grammatik/nomen-und-na-adjektive.md)
**Kanji:** bis WaniKani Level 2 ohne Furigana, alles darüber mit.

---

## Vokabeln für dieses Blatt

**Nomen:** 犬 — Hund · 本 — Buch · 水 — Wasser · 人 — Mensch · 子 — Kind ·
日本 — Japan · 父（ちち）— Vater · 母（はは）— Mutter · 牛（うし）— Kuh ·
先生（せんせい）— Lehrer · 学生（がくせい）— Student · 車（くるま）— Auto

**な-Adjektive:** 元気（げんき）— munter, gesund · すき — mögend · きらい — nicht mögend ·
しずか — leise · きれい — schön, sauber · 上手 — gut (in etwas) · 下手 — schlecht (in etwas) ·
大切（たいせつ）— wichtig

---

## 1. の oder な?

Setz den richtigen Linker ein.

1. 日本 ＿ 車
2. 元気 ＿ 子
3. 先生 ＿ 本
4. しずか ＿ 人
5. 大切 ＿ 本
6. 父 ＿ 車
7. きれい ＿ 水
8. 牛 ＿ 目

## 2. Konjugieren

Bilde alle vier Formen. Achte darauf, was dir dabei auffällt.

| | ist | ist nicht | war | war nicht |
|---|---|---|---|---|
| 学生 | | | | |
| 元気 | | | | |
| 犬 | | | | |
| しずか | | | | |

## 3. だ oder kein だ?

Schreib den Satz so, wie man ihn wirklich sagt. Wo だ hingehört, setz es; wo es
wegfällt, lass es weg.

1. 学生 ＿。 (schlichte Aussage: „Ich bin Student.")
2. 学生 ＿ よ。
3. 元気 ＿。 (schlichte Aussage)
4. 元気 ＿ ね。
5. 犬 ＿ から、…
6. 先生 ＿ った。

## 4. Welche Klasse?

Nomen (N), な-Adjektiv (な) oder い-Adjektiv (い)? Vorsicht — nicht nach dem
letzten Buchstaben gehen.

1. 犬
2. 元気
3. きれい
4. 高（たか）い
5. きらい
6. 上手
7. 水
8. 大切

## 5. Übersetzen

1. Ein munteres Kind.
2. Das Auto aus Japan.
3. Ich bin kein Student.
4. Das Wasser war sauber.
5. Der Lehrer war nicht munter.
6. Ein wichtiges Buch.
7. Vaters Hund.
8. Das Kind war nicht leise.

## 6. Fehler finden

Jeder Satz hat einen Fehler. Finde und korrigiere ihn.

1. 元気の人。
2. きれくない。
3. 日本な車。
4. 学生じゃなかっただ。
