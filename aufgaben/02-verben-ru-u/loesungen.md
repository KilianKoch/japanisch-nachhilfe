# Blatt 2 — Lösungen

> Erst anschauen, wenn du durch bist.

## 1. Wiederholung

| | Wort | verneint | verneinte Vergangenheit |
|---|---|---|---|
| 1 | 犬 | 犬じゃない | 犬じゃなかった |
| 2 | 大切 | 大切じゃない | 大切じゃなかった |
| 3 | 古い | 古くない | 古くなかった |
| 4 | 出る | 出ない | 出なかった |
| 5 | 立つ | 立たない | 立たなかった |

> 1, 2 explizit → じゃない/じゃなかった · 3 い-Adj → くない/くなかった ·
> 4, 5 Verb → ない, und weil ない ein い-Adj ist: なかった.

## 2. Einteilung 一段 / 五段

| | Verb | | Grund |
|---|---|---|---|
| 1 | 上げる | **一段** | げ = e vor る |
| 2 | 立つ | **五段** | endet nicht auf る |
| 3 | 上る | **五段** | ぼ = o vor る |
| 4 | 止める | **一段** | め = e vor る |
| 5 | 引く | **五段** | endet nicht auf る |
| 6 | 広げる | **一段** | げ = e vor る |
| 7 | 入る | **五段** | 例外! sieht aus wie 一段 (はい), ist aber 五段 |
| 8 | 切る | **五段** | 例外! sieht aus wie 一段 (き), ist aber 五段 |

## 3. Vergangenheit (過去形) — 一段

1. 上げる → 上げ**た**
2. 下げる → 下げ**た**
3. 止める → 止め**た**
4. 広げる → 広げ**た**

## 4a. Lücke — は · が · を

1. 犬**が**出た。
2. 子**が**手**を**上げた。
3. 人**が**木**を**切る。
4. 子**が**手**を**下げた。

> 出る hat kein Objekt → nur が. Die anderen: Handelnder が, Objekt を.

## 4b. Was markiert welches Partikel?

| Verb | が | を | は |
|---|---|---|---|
| 出る | den Rausgehenden | nein — kein Objekt | worum es geht |
| 上げる | den, der hebt | **ja** — das Gehobene | worum es geht |
| 立つ | den, der steht | nein — kein Objekt | worum es geht |
| 止める | den, der anhält | **ja** — das Angehaltene | worum es geht |
| 引く | den, der zieht | **ja** — das Gezogene | worum es geht |

> は ist bei allen gleich: es macht das zum Thema, worauf es sitzt — an keine feste
> Rolle gebunden.

## 4c. を oder が? (transitiv / intransitiv)

1. 手**が**上がる — intransitiv (sich heben) → が
2. 火**を**止める — transitiv → を
3. 火**が**止まる — intransitiv (von selbst) → が
4. 手**を**下げる — transitiv → を

> Regel: 他動詞 (transitiv) → を · 自動詞 (intransitiv) → が.

## 4d. が oder は?

1. [Antwort auf „誰が立つ？"] 子**が**立つ。 → が identifiziert
2. [es geht um den Hund] 犬**は**出た。 → は, Thema
3. [wer hob die Hand?] 子**が**手を上げた。 → が identifiziert
4. [Thema: das Kind] 子**は**手を下げる。 → は, Thema

> Frage nach „wer?" → が. Rahmen/Thema gesetzt → は.

## 4e. Fehler finden

1. 手**が**上げる。 → 手**を**上げる。 (上げる transitiv, 手 ist das Objekt → を)
   *(Mit dem intransitiven 上がる wäre 手が上がる auch richtig.)*
2. 子**を**立つ。 → 子**が**立つ。 (立つ hat kein Objekt)
3. 犬**を**出た。 → 犬**が**出た。 (出る hat kein Objekt)

# Interaktiv-Blatt — Antworten

## A. Lesen

1. 犬は大きい？ → ええ、大きい。
2. 犬は目が大きい？ → ええ、大きい。
3. 犬を引く子は大きい？ → 小さい。（犬を引く子は小さい）
4. 手を上げた子は立つ？ → ええ、立つ。
5. 誰が木を切る？ → 父。（木を切る父）

## C. Hören

1. 牛は大きい？ → 小さい。（小さい牛）
2. 牛は目が大きい？ → 小さい。（牛は目が小さい）
3. 牛を引く人は大きい？ → ええ、大きい。
4. 牛が出る？ → ええ、出る。
5. 誰が木を切る？ → 父。（木を切る父）
