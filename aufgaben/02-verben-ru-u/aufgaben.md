## 4. Partikel

### 4a. Lücke füllen — は · が · を

> **Beispiel:** 子 ＿ 火 ＿ 止めた。 → 子**が**火**を**止めた。

1. 犬 ＿ 出た。
2. 子 ＿ 手 ＿ 上げた。
3. 人 ＿ 木 ＿ 切る。
4. 子 ＿ 手 ＿ 下げた。

### 4b. Was markiert welches Partikel?

**が** und **を** hängen an festen Rollen des Verbs. **は** nicht — は hebt nur hervor, **worum es geht**, egal worauf es sitzt.

> **Beispiel** (erste Zeile ist vorgemacht):

| Verb | が | を | は |
|---|---|---|---|
| 出る | den Rausgehenden | nein — 出る hat kein Objekt | worum es geht (z.B. den Hund als Handelnden oder als Thema) |
| 上げる | | | |
| 立つ | | | |
| 止（と）める | | | |
| 引（ひ）く | | | |

### 4c. を oder が? (transitiv / intransitiv)

> **Beispiel:** 手 ＿ 上げる (heben) → 手**を**上げる

1. 手 ＿ 上（あ）がる (sich heben) → 
2. 火 ＿ 止める (etwas anhalten) → 
3. 火 ＿ 止（と）まる (von selbst stehenbleiben) → 
4. 手 ＿ 下げる (senken) → 

### 4d. が oder は? (die stille Frage entscheidet)

> **Beispiel:** [Antwort auf „誰が出た？"] 犬 ＿ 出た。 → 犬**が**出た。

1. [Antwort auf „誰が立つ？"] 子 ＿ 立つ。
2. [Es geht um den Hund — was ist mit ihm?] 犬 ＿ 出た。
3. [Antwort auf „wer hob die Hand?"] 子 ＿ 手を上げた。
4. [Thema: das Kind] 子 ＿ 手を下げる。

### 4e. Fehler finden

> **Beispiel:** 犬**を**出る。 → 出る hat kein Objekt → 犬**が**出る。

1. 手**が**上げる。
2. 子**を**立つ。
3. 犬**を**出た。
