# Teil A — Verben

## 1. Zwei Sorten Verben

**動詞（どうし）** heißt „Verb". Japanisch teilt die 動詞 in zwei Klassen:

- **一段動詞（いちだんどうし）** — die „ru-Verben"
- **五段動詞（ごだんどうし）** — die „u-Verben"

Die Namen sagen, *wie* sie konjugieren: **五段** = „fünf Stufen", das Verb wandert durch alle fünf Vokalreihen (あ・い・う・え・お). **一段** = „eine Stufe", bleibt auf einer.

Erkennen am **Vokal vor dem る**:

| Verb | vor dem る | Sorte |
|---|---|---|
| 見る (み-る) | **i** | 一段 (ru) |
| 出る (で-る) | **e** | 一段 (ru) |
| 上る (のぼ-る) | **o** | 五段 (u) |

**i oder e vor dem る → 一段 (ru). Alles andere → 五段 (u).** Verben ohne る (立つ, 正す, 引く) sind immer 五段.

**Regel zum Dranhalten, kein Naturgesetz:**

> 入（はい）る — endet auf いる, sieht aus wie 一段, ist aber **五段**.

Solche Ausreißer heißen **例外（れいがい）**. Kurze Merkliste, 入る ist die häufigste.

## 2. Vergangenheit (過去形) — nur 一段

Die Vergangenheit heißt **過去形（かこけい）**. Bei 一段-Verben ist sie so leicht wie die Verneinung — **gleiche Mechanik, andere Endung:**

| | Verneinung | 過去形 |
|---|---|---|
| 出る | 出**ない** | 出**た** |
| 上げる | 上げ**ない** | 上げ**た** |
| 止（と）める | 止め**ない** | 止め**た** |
| 広（ひろ）げる | 広げ**ない** | 広げ**た** |

る weg, **た** dran.

**五段 lassen wir heute aus** — deren 過去形 ändert den Stamm über Lautwandel, eigene Stunde.

**Aber die verneinte Vergangenheit ist geschenkt** — für *alle* Verben. Denn ない ist ein い-Adjektiv, und die machen かった:

| | | |
|---|---|---|
| 一段 | 出ない → 出**なかった** | ging nicht raus |
| 五段 | 立たない → 立た**なかった** | stand nicht |

> Kurios: 立た**なかった** geht schon, 立った noch nicht. Die Verneinung ist hier leichter als die Behauptung.

**Damit hast du für 一段 die volle Tabelle:**

| | positiv | negativ |
|---|---|---|
| jetzt | 出る | 出ない |
| 過去形 | 出た | 出なかった |

# Teil B — Die ersten Partikel: は · が · を

## Was ist ein Partikel?

Ein **Partikel** (japanisch **助詞（じょし）**) ist ein kurzes Anhängsel. Es sagt nicht, *was* ein Wort bedeutet, sondern **welche Rolle es im Satz spielt** — wer handelt, an wem gehandelt wird, worum es geht.

Unsere drei heute — **は · が · を** — hängen sich an **Nomen**: an die Mitspieler des Verbs. (Dass Partikel auch an anderem hängen können, kommt später; für は・が・を reichen uns Nomen.)

## Erst die Valenz

Welches Partikel wohin gehört, entscheidet das **Verb**. Also fragt man es zuerst:

> **Valenz** = welche Rollen es bei einem Verb gibt, die ein Partikel markieren kann.

- 出る (rausgehen): **eine** Rolle — wer rausgeht.
- 上げる (heben): **zwei** Rollen — wer hebt, und was gehoben wird.

Kein Muss — im Japanischen kann man jeden Mitspieler weglassen. Es geht nur darum, welche Rollen *da sind*.

Jedes Partikel hat eine **ungefähre Ausrichtung**, welche Rolle es meistens markiert — kein festes Gesetz, aber ein Anhalt. Erst Valenz klären, dann die Partikel drauf.

## が — der Identifizierer

**Allgemein, ohne Verb:** が greift einen Mitspieler heraus und **identifiziert** ihn — „der da, genau der". Den so markierten Mitspieler kann man als den **Handelnden** betrachten, also eine Art **Subjekt** (japanisch **主語（しゅご）**) — das ist eine mögliche Perspektive, kein Muss.

**Der Kern:** が dient immer dazu, etwas zu **identifizieren** — entweder indem es **nach dem X fragt** (誰**が**出る？ — „wer?"), oder indem es ein still mitgedachtes „wer?" **beantwortet** (犬**が**出る — „der Hund").

**An zwei Verben:**

出る (rausgehen) — が stellt die Frage:
> 誰（だれ）**が**出る？ → 犬**が**出る。
> Wer geht raus? → Der Hund.

立つ (stehen) — が beantwortet ein stummes „wer?":
> 子**が**立つ。
> Das Kind ist es, das steht.

## は — das Thema des Kontexts

**Allgemein, ohne Verb:** は sagt nicht, *wer handelt*, sondern **worum es im Kontext geht** — das **Thema**. Innerlich: „**was X angeht, …**". Japanisch: **主題（しゅだい）**.

は spannt den **Rahmen**: „zu diesem Ding jetzt Folgendes". Oft ist das Thema zugleich der Handelnde — dann *sieht* は fast aus wie が, meint aber etwas anderes: Rahmen statt Identifizierung.

**An zwei Verben:**

出る:
> 犬**は**出る。 — Was den Hund angeht: er geht raus.

立つ:
> 子**は**立つ。 — Was das Kind angeht: es steht.

## を — das Objekt (Akkusativ)

**Allgemein, ohne Verb:** を markiert den Mitspieler, **an dem** die Handlung geschieht — das Ding, das getan oder betroffen wird, *nicht* den, der handelt. Das ist das **direkte Objekt**, im Deutschen der **Akkusativ**: „ich hebe **die Hand**". Japanisch: **目的語（もくてきご）**.

を gibt es nur bei **transitiven Verben** (japanisch **他動詞（たどうし）**) — solchen mit direktem Objekt: 上げる, 止める, 引く. Verben ohne Objekt (出る, 立つ) tragen kein を.

**An zwei Verben:**

上げる (heben):
> 手**を**上げる。 — 手 ist, *was* gehoben wird → を.

止（と）める (anhalten):
> 火**を**止める。 — 火 ist, *was* angehalten wird → を.

## Transitiv und intransitiv

Viele Verben gibt es **doppelt** — eine transitive (**他動詞**) und eine intransitive (**自動詞（じどうし）**) Form, oft nur eine Silbe Unterschied:

| transitiv (他動詞) | intransitiv (自動詞) |
|---|---|
| 上げる — etwas heben | 上（あ）がる — steigen, von selbst |

Und da kippt das Partikel:

> 手**を**上げる。 — jemand hebt die Hand. (を: es wird *an* der Hand gehandelt)
> 手**が**上（あ）がる。 — die Hand hebt sich. (が: **kein を mehr** — niemand handelt an ihr, sie tut es selbst)

Beim Wechsel zur intransitiven Form fällt das **を meist weg und wird zu が**: keine direkte Aktion an einem Objekt mehr, nur noch einer, der von selbst etwas tut.

## Überblick: は · が · を

| Partikel | markiert meistens |
|---|---|
| **が** | wer/was etwas macht — **identifiziert** den Handelnden |
| **は** | worum es im Kontext geht — das **Thema** (manchmal zugleich der Handelnde) |
| **を** | woran gehandelt wird — das **Objekt** (Akkusativ) |

> **Nebenbei:** は und が schließen sich nicht aus. Bei *verschiedenen* Mitspielern gehen beide in einem Satz: 子**は**犬**が**出た。 („Was das Kind angeht: der Hund ging raus.") — は steckt den Rahmen, が das Bestimmte darin.
