# Stunde 1 — Inhaltsblatt

Ausführlich: [Die zwei Gruppen](../../grammatik/die-zwei-gruppen.md)

---

## Die Idee

**Man kann japanische Wörter als zwei Gruppen betrachten.** Nicht weil es ein
Naturgesetz wäre — es ist eine Brille. Aber eine, aus der fast alles Weitere von
selbst rausfällt.

**Gruppe A — braucht Hilfe.** 学生（がくせい）ist ein Ding, keine Aussage. Damit es
etwas behauptet, braucht es **だ**.
→ **Nomen** und **な-Adjektive**

**Gruppe B — behauptet sich selbst.** 高（たか）い。und 食（た）べる。sind schon fertige
Sätze. Kein だ.
→ **い-Adjektive** und **Verben**

> Eine Frage, zwei Antworten: *Braucht das Wort ein だ?* ⟺ *Kann es sich selbst verändern?*

## 1. Anschluss ans Nomen

| | | |
|---|---|---|
| **A** Nomen | 日本**の**本 | braucht の |
| **A** な-Adj | 大切（たいせつ）**な**本 | braucht な |
| **B** い-Adj | 高い本 | **nichts** |
| **B** Verb | 食べる人 | **nichts** |

Gruppe A braucht Kleber. Gruppe B klebt selbst.

## 2. Verneinen

| | | |
|---|---|---|
| **A** Nomen | 学生 → 学生**じゃない** | |
| **A** な-Adj | 元気（げんき）→ 元気**じゃない** | immer じゃない |
| **B** い-Adj | 高い → 高**くない** | い → くない |
| **B** Verb (る) | 食べる → 食べ**ない** | る weg, ない dran |
| **B** Verb (う) | 飲（の）む → 飲**ま**ない | う-Reihe → あ-Reihe, ない dran |

**Warum ist A immer じゃない?** Weil gar nicht das Wort verneint wird, sondern das だ
dahinter — und das ist immer dasselbe だ.

**う-Verben:** 飲**む**→飲**ま**ない · 話（はな）**す**→話**さ**ない · 立**つ**→立**た**ない
**Unregelmäßig:** する→しない · 来（く）る→来（こ）ない

## 3. Vergangenheit

| | | |
|---|---|---|
| **A** Nomen | 学生 → 学生**だった** | |
| **A** な-Adj | 元気 → 元気**だった** | immer だった |
| **B** い-Adj | 高い → 高**かった** | い → かった |
| **B** Verb | → **nächste Stunde** | |

Verben lassen wir aus — ihre Vergangenheit braucht Lautwandel (飲む→飲**んだ**,
話す→話**した**, 立つ→立**った**), sechs Muster plus Ausnahme. Eigene Stunde.

## 4. Die だ-Falle

だ liegt unter Gruppe A — aber im blanken Satz **fällt es weg**:

- 学生だ。→ klingt schroff
- 学生。→ so redet man wirklich

Es taucht auf, sobald was dranhängt: 学生**だ**よ。／ 学生**だ**ね。／ 学生**だ**から、…
In じゃない und だった steckt es sowieso schon.

→ だ ist nicht optional. Es ist **die Form unter allem**, nur meistens unsichtbar.

## Aufpassen

- **きれい und きらい sind な-Adjektive**, trotz い am Ende. → きれい**な**人 ✓ /
  きれい**じゃない** ✓ / ~~きれくない~~ ✗ — es geht nach der Klasse, nicht nach der Schrift.
- **~~高いじゃない~~** → 高**くない**。 Gruppe B hat kein だ, also nichts zu verneinen.
- **~~大きいな犬~~** → 大きい犬。 い-Adjektive brauchen keinen Kleber.
- **~~飲まないだ~~** → 飲まない。 Verben haben nie ein だ.

## Vokabeln

**Nomen:** 犬 Hund · 本 Buch · 水 Wasser · 人 Mensch · 子 Kind · 日本 Japan ·
車（くるま）Auto · 父（ちち）Vater · お父（とう）さん Vater (höflich) · 牛（うし）Kuh ·
先生（せんせい）Lehrer · 学生（がくせい）Student · 何（なに）was

**な-Adjektive:** 元気（げんき）munter · すき mögend · きらい nicht mögend ·
しずか leise · きれい schön/sauber · 上手 gut (in etwas) · 大切（たいせつ）wichtig

**い-Adjektive:** 大きい groß · 小さい klein · 高（たか）い teuer/hoch · 古（ふる）い alt ·
正しい richtig

**る-Verben:** 食（た）べる essen · 見（み）る sehen · 出る rausgehen

**う-Verben:** 飲（の）む trinken · 話（はな）す sprechen · 立つ stehen

**Kleinkram:** うん ja · ううん nein · でも aber · これ das hier · それ das da ·
あれ das dort
