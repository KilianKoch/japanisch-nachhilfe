# Vokabeln

Listen nach Thema. WaniKani deckt Kanji und einen guten Teil der Vokabeln ab —
hier steht vor allem, was dort *nicht* vorkommt oder was wir in der Stunde extra brauchen.

## Format

```
漢字（かんじ）— Bedeutung
```

Bei Verben dazu die Gruppe (る/う/unregelmäßig), bei Adjektiven い oder な.
